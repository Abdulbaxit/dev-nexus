# Dev-Nexus: Developer Productivity Hub & Portfolio

[![GitHub Achievement](https://img.shields.io/badge/GitHub-Achievements-blueviolet)](https://github.com/settings/profile)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Dev-Nexus** is a high-performance, aesthetically pleasing developer dashboard designed to streamline daily workflows. Built with a focus on modern UI/UX principles like **Glassmorphism**, it integrates real-time GitHub profile data and productivity tools.

![Dev-Nexus Preview](https://via.placeholder.com/1200x600/1e293b/f8fafc?text=Dev-Nexus+Premium+Dashboard)

## 🚀 Key Features

- **GitHub Profile Sync**: Leverages the **GitHub REST API** to dynamically fetch and display user profiles, avatars, and bios.
- **Persistent Task Management**: A custom-built task tracker with **LocalStorage** persistence, allowing developers to track progress across sessions.
- **Dynamic News Feed**: An interactive feed showcasing technology headlines and developer updates.
- **Aesthetic UI/UX**:
  - Full **Dark/Light Mode** support with smooth CSS transitions.
  - Responsive **Glassmorphism** design using backdrop filters and CSS custom variables.
  - Interactive data visualizations with CSS-driven animations (Contribution Graph).
- **SEO & Performance**: Optimized for fast load times and clean semantic HTML.

## 🛠️ Technology Stack

- **Frontend**: ES6+ JavaScript, CSS3 (Modern Flexbox/Grid), Semantic HTML5.
- **APIs**: GitHub REST API for profile synchronization.
- **Typography**: Inter & Outfit via Google Fonts for a premium look.
- **Optimization**: LocalStorage for state persistence, Backdrop Blur for advanced UI effects.
