# Dev-Nexus Dashboard

A sleek, professional productivity hub designed for modern developers.

![Dev-Nexus Preview](https://via.placeholder.com/1200x600/1e293b/f8fafc?text=Dev-Nexus+Workspace+Dashboard)

## Features

- 🚀 **Personalized Workspace**: Glassmorphism UI with dynamic greetings and theme support.
- 📋 **Integrated Task Tracker**: Manage your daily development velocity with a built-in interactive list.
- 📰 **Developer News Feed**: Real-time mock feed of top technology headlines.
- 📊 **Contribution Insights**: Visual representation of your development activity and git metrics.
- 📱 **Fully Responsive**: Optimized for desktops, tablets, and mobile devices.

## Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Variables, Animations, Glassmorphism)
- **Logic**: ES6+ Javascript
- **Typography**: Inter & Outfit (Google Fonts)

## Getting Started

Simply open `index.html` in your browser to access your local dashboard. No build steps required.

```bash
git clone https://github.com/YOUR_USERNAME/dev-productivity-hub.git
cd dev-productivity-hub
open index.html
```
